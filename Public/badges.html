Badges

    Cette page decrit comment debloquer chaque badge.

* Les badges montrés ici sont les badges qui peuvent être débloqués. Certains badges ne seront attribués que par l'Admin pour des raisons spéciales. D'autres badges apparaîtront sûrement ici au fur et à mesure, ouvre l'oeil.

⭐​ EasterEgg S[?]

Les badges EasterEgg sont des récompenses données pour avoir découvert des un Easter Egg.

Vous pouvez voir votre avancement sur les EE dans la page EE Tracker disponible depuis la sidebar.

💻​ SLAM

Ce badge est donné aux étudiants en SLAM.

Si vous n'avez pas de badge d'Option, envoyez un MP à l'Admin avec votre Option.

🔐 SISR

Ce badge est donné aux étudiants en SISR.

Si vous n'avez pas de badge d'Option, envoyez un MP à l'Admin avec votre Option.

📢​ Orateur

Ce badge est donné aux utilisateurs qui ont réalisé un bon oral dans n'importe quelle matière avec une certaine facilité à l'oral.

Les critères de qualité de l'oral sont gardés par l'Admin et seul lui décidera de l'obtention du badge.

🖱️​ Clicker Fou

Ce badge est donné aux utilisateurs qui ont une grande activité de clics et à ceux qui aiment cliquer (pour rien).

Un CPS humain maintenu à 14 pendant 6.7 secondes est requis pour obtenir ce badge.

1️⃣​ Uno Addict

Ce badge est donné aux utilisateurs qui ne peuvent pas se passer du jeu Uno.

Il vous faudra un minimum de 50 parties jouées pour obtenir ce badge.

💁‍♀️​​ Lady

Ce badge est donné aux femmes. (obviously)

Le badge ne sera pas donné au hommes et gars qui veulent devenir trans.

🤍​ OG

Ce badge est donnés aux utilisateurs ayant créé un compte avant les vacance de Noël 2025.

Si vous pensez que votre compte est assez vieux, contactez un Admin pour vérification de l'égibilité.

🌟​ Champion d'Antan S[?]

Ce badge est donné en fin de Saison à l'utilisateur qui compte le plus de TOP1 dans les leaderboards.

Bonne chance pour maintenir vos titres.

🕵️‍♂️ Bug Finder

Ce badge est donné aux utilisateurs qui signalent des bugs.

Un petit bug ne vous donnera pas le badge. Il vous faudra signaler plusieurs petits bugs ou un seul bug majeur pour l'avoir.

🍀​ Casino Master

Ce badge est donné aux utilisateurs qui ont une grande chance au casino et aux gros dépenseurs dans les jeux d'argents.

Un minimum de 100 000 C dépensés dans les jeux d'argent est requis pour obtenir ce badge.

📝 Motus Fanatic

Ce badge est donné aux utilisateurs qui aime se creuser la tête sur le jeu Motus.

Un minimum de 45 mots trouvés est requis pour obtenir ce badge.

🧩 2048 Master

Ce badge est donné aux utilisateurs qui empilent les tuiles dans 2048.

Il vous faudra atteindre une tuile 1024 ou un score de 5000 pour obtenir ce badge.

🦖 Dino Runner

Ce badge est donné aux utilisateurs qui performent sur le jeu Dino.

Un score minimum de 7500 est requis pour obtenir ce badge.

🐤 Flappy Pilot

Ce badge est donné aux utilisateurs qui maîtrisent le jeu Flappy.

Un score minimum de 100 est requis pour obtenir ce badge.

🐍 Snake Legend

Ce badge est donné aux utilisateurs qui font de très gros scores sur Snake.

Un score minimum de 120 est requis pour obtenir ce badge.

🧱 Block Blast Pro

Ce badge est donné aux utilisateurs qui enchaînent les blocs sur Block Blast.

Un score minimum de 10000 est requis pour obtenir ce badge.

🔢 Sudoku Solver

Ce badge est donné aux utilisateurs qui complètent régulièrement des grilles Sudoku.

Un minimum de 20 grilles complétées est requis pour obtenir ce badge.

🔴 Puissance 4 Champion

Ce badge est donné aux utilisateurs qui gagnent souvent à Puissance 4.

Un minimum de 15 victoires est requis pour obtenir ce badge.

⚡ Mash Dominator

Ce badge est donné aux utilisateurs qui dominent les parties de Mash.

Un minimum de 15 victoires est requis pour obtenir ce badge.

🪙 Coinflip Addict

Ce badge est donné aux utilisateurs qui enchaînent les parties de Coinflip.

Un minimum de 50 parties jouées est requis pour obtenir ce badge.

🃏 Blackjack Grinder

Ce badge est donné aux utilisateurs qui jouent souvent des mains de Blackjack.

Un minimum de 50 mains jouées est requis pour obtenir ce badge.

🎯 Roulette Veteran

Ce badge est donné aux utilisateurs qui enchaînent les tours de Roulette.

Un minimum de 40 parties jouées est requis pour obtenir ce badge.

🎰 Slots Spinner

Ce badge est donné aux utilisateurs qui jouent régulièrement aux Slots.

Un minimum de 40 parties jouées est requis pour obtenir ce badge.
